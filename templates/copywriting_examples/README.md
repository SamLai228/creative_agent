# 壽險文案範例說明

## 檔案位置

`templates/copywriting_examples/examples.json`

## 如何提供公司文案範例

### 方式一：直接編輯 JSON 檔案

編輯 `examples.json` 檔案，在 `examples` 陣列中加入您的壽險文案範例：

```json
{
  "examples": [
    {
      "title": "您的標題範例",
      "content": "您的內文範例",
      "category": "險種分類（如：終身壽險、醫療險、意外險、重大疾病險、癌症險、儲蓄險、投資型保單等）",
      "tone": "風格（如：溫馨、專業、安心、關懷、穩健等）",
      "target_audience": "目標受眾（如：家庭、一般消費者、中壯年、年輕族群、有儲蓄需求者等）"
    }
  ]
}
```

### 方式二：使用 API 端點（未來可擴展）

目前可以手動編輯 JSON 檔案，未來可以擴展為透過 API 上傳範例。

## 欄位說明

- **title**: 標題範例（8-15 個字為佳）
  - 應傳達保障、安心、專業的價值
  - 可強調險種特色或保障內容
- **content**: 內文範例（30-80 個字為佳）
  - 說明險種特色、保障內容或優惠方案
  - 適合疊在圖片上顯示（不要太長）
- **category**: 險種分類（可選）
  - 如：終身壽險、定期壽險、醫療險、意外險、重大疾病險、癌症險、儲蓄險、投資型保單等
- **tone**: 文案風格（可選）
  - 如：溫馨、專業、安心、關懷、穩健、實惠等
  - 符合壽險產業特色
- **target_audience**: 目標受眾（可選）
  - 如：家庭、一般消費者、中壯年、年輕族群、有儲蓄需求者、有投資需求者等

## 範例數量建議

- 建議提供 **5-10 個**不同險種的範例
- 涵蓋不同的險種類型、風格和目標受眾
- 範例越多，生成的文案越符合公司風格

## 壽險文案特色

- 強調保障價值、安心感、專業服務
- 傳達對客戶的關懷和守護
- 避免過度銷售話術
- 以專業和溫馨的語調為主

## 注意事項

- 使用繁體中文
- 標題要簡潔有力，傳達保障價值
- 內文要適合疊在圖片上顯示（不要太長）
- 保持公司品牌語調的一致性
- 符合壽險產業的專業形象
